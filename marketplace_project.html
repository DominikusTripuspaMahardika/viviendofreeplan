Aplikasi Jasa Salon (Codeigniter)

Aplikasi Jasa Salon adalah sebuah sistem informasi pelayanan jasa yang dibuat dan digunakan untuk memudahkan customer dalam memesan jasa salon secara online. Aplikasi ini dilengkapi dengan fitur login multi user, register user, data layanan, data galeri dan lain sebagainya.

Sistem Requirement:

  • Bahasa: PHP 7.4
  • Database: MySQL
  • Server: XAMPP

Panduan Instalasi:

  1. Akses aplikasi via localhost/salon.
  2. Login Admin: admin | Password: admin.

Aplikasi Pemesanan Kopi (Codeigniter)

Aplikasi Pemesanan Kopi adalah sebuah sistem informasi pemesana kopi yang dibuat dan digunakan untuk memudahkan pelanggan dalam memesan kopi secara online. Aplikasi ini dilengkapi dengan fitur login multi user, katalog kopi, transaksi pemesanan, data pelanggan dan lain sebagainya.

Sistem Requirement:

  • Bahasa: PHP 7.4
  • Database: MySQL
  • Server: XAMPP

Panduan Instalasi:

  1. Akses aplikasi via localhost/pengaduan.
  2. Login Admin: admin | Password: admin.
  3. Login Pemiliki: pemilik | Password: pemilik.
